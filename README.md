## 🔧 Quick Use Guide

1. **Download** the whole `.zip` from `/fonts`.
2. **Install** on:
   ```text
   • Windows — Right-click ▸ Install
   • macOS   — Double-click ▸ Install Font
   • Linux   — ~/.fonts + fc-cache -f -v
   • Android — Copy to /system/fonts (root) or use iFont
